Para agregar grupos al sistema en Linux, utilizo el comando groupadd, que es la herramienta que se usa para crear nuevos grupos. Un grupo en Linux es simplemente un conjunto de usuarios que pueden compartir ciertos permisos en archivos o directorios, lo cual ayuda a gestionar el acceso a recursos de manera más eficiente. Para agregar un grupo, simplemente ejecuto el comando groupadd seguido del nombre del grupo que quiero crear. Por ejemplo, si quiero crear un grupo llamado "desarrolladores", simplemente escribo sudo groupadd desarrolladores. Esto crea el grupo en el sistema. Una vez creado, puedo agregar usuarios a ese grupo, lo cual se hace con el comando usermod. Para añadir un usuario a un grupo sin quitarlo de otros grupos a los que ya pertenezca, uso usermod -aG. Por ejemplo, para agregar el usuario "juan" al grupo "desarrolladores", escribiría sudo usermod -aG desarrolladores juan. De esta forma, puedo organizar a los usuarios en grupos y gestionar de manera más sencilla los permisos de acceso a los recursos del sistema.
